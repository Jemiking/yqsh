## 数据结构说明

每个任务条目包含：
- `id`: 任务唯一标识符
- `task`: 任务名称
- `task_en`: 任务英文名
- `description`: 任务详细描述
- `priority`: 优先级 (HIGH / MEDIUM / LOW)
- `category`: 分类 (MEDICAL / PREPARATION / EMOTIONAL / NUTRITION / PHYSICAL)
- `deadline`: 完成期限提示

## 使用说明

### 任务推送逻辑

1. **根据当前孕周**获取对应周的任务列表
2. **按优先级排序**：HIGH > MEDIUM > LOW
3. **检查deadline**判断是否需要提醒
4. **每日推送**：优先推送未完成的HIGH优先级任务

### APP交互示例

**第24周早晨推送**:
```
📅 第24周第1天

本周重点任务：
✓ 报名分娩课程 [HIGH]
✓ 组装婴儿床和衣柜 [MEDIUM]

```
